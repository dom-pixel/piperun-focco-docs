Getting Started

Warning: To proceed, you need to have the Schuster extension installed on your browser!

To get access to the API, login to your piperun account and get your token, here is a simple way to do it:

  • 1: Open Chrome DevTools on the piperun website, you need to be logged to work, otherwise it will return a error.

  • 2: Go to Console and paste this code:

    sessionStorage.pipeToken

  • 3: Copy the Token from the console (dont copy the apostrophes)

  • 4: Put on the headers like this:

  • 5: Now you can make requests to the API!
